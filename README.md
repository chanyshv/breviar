URL shortener cli.
## Installation
```shell script
pip3 install breviar
```

## Usage 
```
Usage: breviar [OPTIONS] COMMAND [ARGS]...

Options:
  --help  Show this message and exit.

Commands:
  configure  Configure a service
  shorten    Shorten a link
```

## Usage examples

**Configure service**
```shell script
breviar configure
```
**Shorten a link**
 ```shell script
breviar shorten https://russkie_vpered.com
```
